# ubi-bagels

Ubi-bagels is a web app that will remind you that free bagels are available. That is, every last Friday of the month.

## Installation

To start the server, simply run `npm start` from the terminal.

## Usage

When the server is up, a web page is server, which explains how to use the service.
Simply put, you can send an SMS to the specified number to register to the notifications.
Sending STOP will cancel the subscription.
